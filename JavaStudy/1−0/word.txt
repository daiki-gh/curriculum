■ SQL
 SQL　リレーショナルデータベース管理システム
  DB2　IBMのリレーショナルデータベース管理システム
  MySQL　Oracleのオープンソースのリレーショナルデータベース管理システム
   Maria DB　MySQLから派生したオープンソースのリレーショナルデータベース管理システム
  Oracle DB　Oracleのデータベース管理システム
  PostgreSQL　RDBの作成や操作、管理ができるオープンソースのDBMS
  SQLite　PHPと組み合わせて使用されるDBMS
  SQL Server　SQLのRDBS
 CRUD　Create生成、Read読み込み、Update更新、Delete削除の４つの基本機能
 トランザクション　お互いに関連のある複数の処理をまとめて行い一つの処理とすること
  コミット　結果を確定させる
  ロールバック　トランザクション処理を戻す
 排他制御（排他ロック）　共有資源に対して同時にアクセスしても問題なく動作できる制御のこと
  楽観ロック　そもそも「同時アクセスはあまり起こらない」という楽観的な考え方、更新対象のデータが更新前と同じであるかを確認することでデータの整合性を保つ排他制御
  悲観ロック　同じデータを他者が頻繁に更新するという悲観的な考え方、更新データを取得時にロックして他者が操作できないうようにする方式
 実行計画　どうしたらより短い時間で実行できるかを計算して導きだした計画
 チューニング　調整
 パフォーマンス　成果
 スロークエリ　RDBSに対するクエリに時間がかかっているもの
 クエリログ　問い合わせの記録
 IDE（統合開発環境）　作るために必要なものが全部詰まった開発環境
  Eclipse　オープンソースで開発されているソフトウェア統合開発環境
  NetBeans　プロジェクト管理機能や入力支援機能を備えたオープンソースのIDE
  SpringToolSuite　Eclipseベースの無償のIDE
 DBツール　DBの操作方法
  A5:SQL Mk-2　データベース開発の生産性を向上させるためのSQLツール
  pgAdmin　PostgreSQLの管理ツール、遠隔のデータベースサーバをグラフィカルな表示・操作体系で管理できる
  DBeaver　SQLクライアント・DB管理ツール
  MySQL Workbench　包括的な管理ツール
  Sequel Pro　Mac用のMySQLのデータをGUI操作で管理するためのアプリケーション
  TeamSQL　SQLクライアント・DB管理ツール
